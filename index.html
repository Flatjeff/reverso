Comment jouer à REVERSO ?

Phase 1 — Écoute

Le joueur actif appuie sur ÉCOUTER. Le Game Master prononce un mot à l'envers. Tu peux le réécouter jusqu'à 3 fois.

Phase 2 — Répétition

Le joueur tente de répéter le son qu'il a entendu. Il enregistre sa tentative (1 seul essai).

Phase 3 — Devinette

L'enregistrement est rejoué à l'envers. Chaque joueur peut faire une seule proposition.

Phase 4 — Solution

La solution est révélée. 1 jeton pour le joueur actif si quelqu'un a trouvé. 1 jeton pour celui qui a deviné. Premier à 5 jetons gagne.
